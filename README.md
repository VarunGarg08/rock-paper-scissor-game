🎮 Rock Paper Scissors Game (Python + Tkinter)

A simple Rock Paper Scissors game built using Python and the Tkinter library.
This project is created to practice basic Python programming and GUI development.

The player can choose Rock, Paper, or Scissors, and the computer randomly selects one option.
The program then compares both choices and shows the result on the screen.

---

🎮 How the Game Works

1. The player clicks one of the buttons: Rock, Paper, or Scissors.
2. The computer randomly chooses an option.
3. The program compares both choices.
4. The result (Win / Lose / Tie) is displayed.
5. The score updates after each round.

---

🎯 Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

If both the player and the computer choose the same option, the round ends in a Tie.

---

🖥️ Interface

The game interface contains:

- Game title
- Player choice display
- Computer choice display
- Result message
- Scoreboard
- Three buttons for Rock, Paper, and Scissors

The interface is built using the Tkinter GUI toolkit.
